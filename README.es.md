# Hechicero de Despliegue

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad de Hechicero de Despliegue para OpenClaw.

## Qué Soluciona
Despliega y gestiona automáticamente la infraestructura de aplicaciones con configuración mínima.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque de seguridad primero
- Soporte para rollback

## Uso
/deployment-sorcerer

## Ejemplo
```
Analizar código para problemas de DevOps.
```

Esta habilidad está automáticamente disponible en OpenClaw.
